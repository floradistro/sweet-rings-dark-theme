# Sweet Rings Donut Co. - Dark Theme Website

A modern, responsive website for Sweet Rings Donut Co. featuring a sleek dark theme design.

## Features

- **Dark Theme Design**: Professional dark color scheme with orange accents
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Modern UI**: Clean, modern interface with smooth animations
- **Interactive Elements**: Hover effects and smooth scrolling
- **Menu Showcase**: Beautiful display of donut offerings with pricing
- **Contact Information**: Store hours, location, and ordering options

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- JavaScript
- Google Fonts (Poppins)

## Color Scheme

- **Background**: #121212 (Primary dark)
- **Secondary Background**: #1a1a1a
- **Cards/Elements**: #2d2d2d
- **Text Primary**: #e0e0e0
- **Text Secondary**: #b0b0b0
- **Accent Color**: #ff6b6b (Orange-red)
- **Borders**: #333

## Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. Or run a local server:
   ```bash
   npm start
   ```

## Development

To run the development server:
```bash
npm run dev
```

## Sections

- **Hero**: Main landing section with call-to-action
- **Features**: Key selling points (Fresh Daily, Premium Ingredients, Fast Delivery)
- **Menu**: Signature donut showcase with prices
- **About**: Company story and statistics
- **Contact/Order**: Location info and ordering options

## Responsive Breakpoints

- Desktop: 1200px+
- Tablet: 768px - 1199px
- Mobile: 480px - 767px
- Small Mobile: <480px
